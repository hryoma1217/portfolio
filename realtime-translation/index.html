音声を、見ながら読む。Realtime Translation — リアルタイム字幕翻訳

PC スピーカーから流れる音声をリアルタイムで文字起こしし、翻訳して画面に字幕としてオーバーレイ表示する Windows アプリ。 ゲームや海外配信を、画面を切り替えずにそのまま理解できます。

Python WASAPI ループバック録音 Whisper STT Groq / OpenAI / Gemini / Ollama 透過オーバーレイ PyInstaller 配布

01できること

システム音声をそのまま翻訳

WASAPI ループバックで PC の再生音声を直接取得。マイク不要で、動画・ゲーム・配信の音をそのまま字幕化します。

画面に字幕オーバーレイ

常に最前面の透過ウィンドウに表示。位置・フォントサイズ・背景色を調整でき、全画面ゲーム上でも読めます。

翻訳エンジンを選べる

Groq（無料枠）/ OpenAI / Anthropic / Gemini / ローカル Ollama に対応。無料で始めて、必要なら高精度へ切替。

ノイズ・混在音声に強い

RMS 無音閾値とスピーカーフィルタで BGM や日本語音声を除外。リアクション動画の英語コメントだけを拾えます。

02処理の流れ

システム音声取得→ 無音/話者フィルタ→ Whisper 文字起こし→ AI 翻訳→ 字幕オーバーレイ

032つの動作モード

モードB — 常時翻訳

音声が流れている間つねに翻訳。映画・動画視聴やゲーム観戦向け。

モードA — 操作連動

キーボード・マウス操作中だけ翻訳し、API 使用量を抑制。プレイ中心の利用向け。

04技術的なポイント

低遅延パイプライン

録音・STT・翻訳・描画を別スレッド/ワーカーで非同期化し、字幕が遅れて積み上がらないよう制御。

プライバシー配慮

ローカル STT 時は音声を外部送信せず PC 内で完結。クラウド利用時の送信範囲も明示。

ワンファイル配布

PyInstaller で exe 化し、トレイ常駐・右クリック設定で IT が苦手な人でも使える導線に。

文脈ヒント注入

作品名・キャラ名を渡して固有名詞の訳ブレを低減。アニメ・ゲーム実況の精度を底上げ。
