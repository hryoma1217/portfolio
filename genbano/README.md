# Genbano — Portfolio Case Study Page

このフォルダは、建設業向け業務統合SaaS「ゲンバノ (Genbano)」を匿名化したケーススタディとして紹介する、GitHub Pages 用のポートフォリオページ（単一の `index.html`）です。外部JS/CSS/CDNに依存せず、すべてのスタイルを `<style>` にインライン化しているため、リポジトリに置いて GitHub Pages を有効化するだけでそのまま表示できます。実在のクライアント名・企業名・個人名・ドメイン・シークレットは一切含みません。公開するには、このフォルダ（または `index.html`）をGitHubリポジトリへ push し、リポジトリの Settings → Pages で対象のブランチ（例: `main`）とフォルダ（ルート、または `/portfolio/genbano` を含めたい場合は `/docs` などへ配置）を指定して Pages を有効化してください。数十秒後に発行されたURLでページが閲覧できます。
